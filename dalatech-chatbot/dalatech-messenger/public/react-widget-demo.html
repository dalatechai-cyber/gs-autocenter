🚀 DalaTech.ai React Chat Widget

Welcome to the demonstration of the DalaTech.ai React-powered floating chat widget. This is a modern, fully-featured chatbot interface built with React and Tailwind CSS.

Features:

  • Floating button in bottom-right corner
  • Expand/collapse chat window on click
  • Real-time message exchange with backend API
  • Loading indicator during API calls
  • Conversation history persistence
  • Error handling for failed requests
  • Styled with Tailwind CSS
  • Responsive design (mobile & desktop)
  • Default collapsed state on page load
👉 Look at the bottom-right corner!
Click the chat button to start a conversation with DalaTech.ai's AI assistant.
