# Privacy Policy & Data Handling

## DalaTech.ai Chatbot Privacy Policy

**Last Updated**: January 2026

### 1. Information We Collect

#### Conversation Data
- Messages you send to the chatbot
- Conversation history (last 20 messages)
- Timestamps of interactions
- Response times and performance metrics

#### Technical Data
- IP address (for rate limiting and security)
- Browser type and version
- Device type
- Request metadata

### 2. How We Use Your Information

#### Primary Purposes
- **Service Delivery**: Provide AI-powered customer service
- **Improvement**: Analyze conversation patterns to improve responses
- **Security**: Prevent abuse and malicious activity
- **Analytics**: Track usage metrics and system performance

#### We Do NOT
- ❌ Sell your data to third parties
- ❌ Use data for advertising purposes
- ❌ Share conversations with unauthorized parties
- ❌ Store payment information (we don't process payments)

### 3. Data Storage

#### Local Storage (Your Device)
- Conversation history: Stored in browser LocalStorage
- Duration: Until you clear browser data or clear chat
- Control: You can clear this anytime using the trash button

#### Server Storage
- **Temporary**: Messages processed in real-time
- **Logging**: Request IDs and performance metrics
- **Analytics**: Aggregated, anonymized usage statistics
- **Retention**: Logs retained for 30 days for troubleshooting

### 4. Data Security

#### Security Measures
✅ HTTPS encryption for all communications
✅ Input validation and sanitization
✅ Rate limiting to prevent abuse
✅ No sensitive data stored in client-side code
✅ API keys secured in environment variables

#### Your Responsibility
- Don't share sensitive personal information (passwords, credit cards)
- Use secure networks when accessing the chatbot
- Clear chat history if using a shared device

### 5. Third-Party Services

#### Google Gemini AI
- **Purpose**: AI response generation
- **Data Shared**: Your messages and conversation context
- **Privacy**: Subject to [Google's Privacy Policy](https://policies.google.com/privacy)
- **Note**: We don't control how Google processes data

#### Vercel Hosting
- **Purpose**: Application hosting and delivery
- **Data Shared**: Technical/server logs
- **Privacy**: Subject to [Vercel's Privacy Policy](https://vercel.com/legal/privacy-policy)

### 6. Your Rights

You have the right to:
- **Access**: Request information about data we process
- **Deletion**: Clear your conversation history anytime
- **Correction**: Contact us to correct inaccurate information
- **Opt-out**: Stop using the service at any time

### 7. Data Retention

| Data Type | Retention Period | Storage Location |
|-----------|------------------|------------------|
| Conversation History | Until cleared by user | Browser LocalStorage |
| Request Logs | 30 days | Server logs |
| Analytics | 90 days (aggregated) | Server memory |
| Feedback | Indefinite (anonymized) | Server memory |

### 8. Children's Privacy

This service is intended for business use. We do not knowingly collect information from children under 13.

### 9. Changes to This Policy

We may update this policy periodically. Check the "Last Updated" date above. Continued use after changes constitutes acceptance.

### 10. Contact Us
